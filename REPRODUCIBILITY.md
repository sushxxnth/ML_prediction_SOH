# Reproducibility Guide

This document provides step-by-step instructions to reproduce every quantitative claim in the manuscript. All results shown here are computed by running model inference or reading files generated by training scripts — no values are hardcoded.

---

## Step 0: Environment Setup

```bash
# Python 3.8+ required
pip install torch numpy pandas matplotlib scikit-learn scipy

# Download pre-trained weights (~1 MB, hosted on GitHub Release v1.0.0)
python3 download_weights.py
```

This installs model weights into `reports/` (gitignored). The script requires the [GitHub CLI](https://cli.github.com/) to be authenticated (`gh auth login`).

## Step 2: Individual Claim Verification

### Claim 1 — Causal Attribution Accuracy: 96.0% (95% CI: 90.7–100.0%)

```bash
python3 VERIFY_96_ACCURACY.py
```

**How it works:** Loads `reports/pinn_causal/pinn_causal_retrained.pt` and runs a forward pass on all 75 benchmark scenarios (NASA, Panasonic EV, Nature MATR, Randomized 40°C, HUST LFP). Compares predicted dominant mechanism against condition-based ground truth labels (validated against post-mortem ICA/DVA data where available).

Expected output:
```
NASA        : 14/15 ( 93.3%)
Panasonic   : 15/15 (100.0%)
Nature      : 15/15 (100.0%)
Randomized  : 14/15 ( 93.3%)
HUST        : 14/15 ( 93.3%)
Overall     : 72/75 ( 96.0%)
SUCCESS: 96% accuracy threshold achieved!
```

The 95% bootstrap confidence interval [90.7%–100.0%] was computed via 10,000-resample bootstrap on the 72/75 binary outcomes using `numpy.random.default_rng(42)`.

### Claim 2 — HERO Zero-Shot RUL (MAE 44.0 cycles, 55% reduction)

```bash
python3 verify_hero_zeroshot.py
```

**How it works:** Evaluates HERO on TJU NCA cells (unseen during training). Compares against LSTM baseline from `reports/zeroshot_baseline_comparison.json`. Reduction is computed as `(LSTM_MAE - HERO_MAE) / LSTM_MAE × 100`.

### Claim 3 — Ablation: Expert Priors Add 18.7 Percentage Points

python3 verify_ablation_comprehensive.py

## Dataset Notes

The framework uses two **distinct, non-overlapping** dataset groups:

| Purpose | Datasets | Cells |
|---------|----------|-------|
| HERO memory bank (prediction) | NASA, CALCE, Oxford, TJU, XJTU | 76 |
| Attribution benchmark (75 scenarios) | NASA, Panasonic EV, Nature MATR, Randomized, HUST LFP | 75 scenarios |

All datasets are publicly available. See the **Datasets** section of `README.md` for download links.

---

## Ground-Truth Label Transparency

The 75 attribution benchmark scenarios were assigned condition-based labels following established electrochemical rules (e.g., cells below 10°C at ≥1C → Lithium Plating dominant). For the 23 scenarios with published post-mortem or ICA/DVA data, condition-based labels agreed with experimental evidence in 22/23 cases (96%).

This labeling methodology is described in detail in Section 5.2 of the paper.

---

## Known Failure Cases

The 3 misclassified scenarios (out of 75) all involve high-temperature, high-discharge conditions (43°C/2C) where both SEI growth and Active Material Loss are simultaneously thermally activated. This is the residual disambiguation challenge discussed in Section 4.3.
